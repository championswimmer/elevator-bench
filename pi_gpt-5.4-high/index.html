Elevator Simulator

Model: gpt-5.4:high · Tool: pi · Provider: github

A 10-floor building with 4 elevators. Hall calls prefer the closest idle elevator, busy periods queue requests, and each elevator finishes its current travel direction before reversing.

Default configuration

  • Floors: 10 (0–9)
  • Elevators: 4 (E0–E3)
  • Serve port: 8808

How to use

  • Use floor ↑/↓ buttons to create hall calls.
  • Use elevator panels to choose destinations inside each car.
  • Watch queued and assigned requests update in real time.
